# Airbnb Market Analysis Dashboard | Power BI

## 1. Project Title / Headline

**Airbnb Listing Performance & Market Insights Dashboard using Power BI**

---

## 2. Short Description / Purpose

This project focuses on analyzing Airbnb listing data to uncover trends in pricing, availability, customer reviews, and property performance. The dashboard provides a comprehensive view of the short-term rental market, helping stakeholders understand factors that influence listing success and guest engagement.

The objective was to transform raw Airbnb data into meaningful insights through interactive visualizations, enabling data-driven decisions related to pricing strategy, host performance, and market opportunities.

### Listing Performance Analysis
